Home page

This mini-app that provides a simplified mission planning capability, empowering operators to create and visualise robot movement. This means that the operator should be able to create a list with an arbitrary number of 2D points and watch the robot navigate them in a dedicated visualisation area.

App Structure

The app is structured into 3 main components:

  • Home: A landing page that provides information about the app.
  • Planner: Allows operator to plan the route for the robot to perform an inspection adding new points in table .
  • Player: Operator can visualize the route that the robot will valk to make sure all the points are correct.

Running the App Locally

Before running the app, make sure you have the following installed:

  • Node.js
  • Angular CLI
  • Angular Material

Follow these steps to get the app running on your local machine:

  1. Clone the repository: git clone https://github.com/DanielTomaCucu/mission-planner.git
  2. Navigate to the project directory: cd mission-planner
  3. Install dependencies: npm install
  4. Start the app: ng serve
  5. Open your browser and visit http://localhost:4200/
